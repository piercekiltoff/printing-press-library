# Mercury CLI Brief

Mercury has an official Go CLI, currently useful for human operators. The Printing Press version proceeds because it adds agent-native affordances the official CLI does not provide: MCP exposure, machine-readable command context, local archive/search, universal dry-run behavior, and a generated money-movement planning workflow.

The official Mercury CLI remains the human-polish benchmark. The printed CLI should be judged on whether an agent can discover commands, form safe write plans, inspect local context, and avoid accidental money movement.
