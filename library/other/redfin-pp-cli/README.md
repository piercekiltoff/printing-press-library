# redfin-pp-cli

Redfin real estate CLI with offline search, market analysis, and portfolio tracking.

Search properties, track valuations, analyze markets, and manage a real estate portfolio from the terminal. All data persists locally in SQLite for offline access and cross-entity queries.

## Install

```
go install github.com/mvanhorn/printing-press-library/library/other/redfin-pp-cli/cmd/redfin-pp-cli@latest
```

## Quick Start

```bash
# Search for properties
redfin-pp-cli search "San Francisco, CA" --min-price 500000 --beds 3

# Get property details
redfin-pp-cli property info /CA/San-Francisco/123-Main-St/home/12345

# Calculate mortgage
redfin-pp-cli mortgage 750000 --down 20 --rate 6.5

# Track market trends
redfin-pp-cli pulse "Austin, TX"

# Build a portfolio
redfin-pp-cli portfolio add 12345 --label watching --alert-below 500000
```

## Commands

### Search and Discovery

- `search <location>` - Search properties by location with filters (price, beds, baths, sqft, type)
- `deals <location>` - Find properties priced below their AVM estimate
- `stale --region <location>` - Find listings with high days on market

### Property Analysis

- `property info <url-or-id>` - Property overview (price, beds, baths, photos)
- `property details <id>` - Full property details (amenities, history)
- `property value <id>` - Redfin valuation estimate
- `property comps <id>` - Comparable properties (active or sold)
- `property nearby <id>` - Nearby properties
- `property neighborhood <id>` - Walk, bike, and transit scores
- `property costs <id>` - Cost of ownership breakdown
- `property commute <id>` - Commute information
- `property history <id>` - Price history from local database

### Financial Tools

- `mortgage <price>` - Calculate monthly payments with tax/insurance estimates
- `mortgage <price> --compare` - Side-by-side 15yr vs 30yr comparison
- `invest <id> --rent 3500` - Investment metrics (cap rate, cash-on-cash, GRM)
- `score <id>` - Score properties against custom weighted profiles

### Market Intelligence

- `pulse <region>` - Market snapshot (median price, inventory, temperature)
- `trends <region>` - Market trends and statistics
- `trends compare <region1> <region2>` - Side-by-side trend comparison
- `compare-hoods <hood1> <hood2>` - Neighborhood comparison
- `analyze-zips <zip1> <zip2> ...` - Cross-zip code analysis
- `report <region>` - Generate a full market report

### Portfolio and Tracking

- `portfolio` - Dashboard of all tracked properties
- `portfolio add <id>` - Add a property to track
- `portfolio alerts` - Check price thresholds
- `track <id>` - Track property valuation trajectory over time

### Data

- `data list` - Available Redfin Data Center datasets
- `data download <dataset>` - Download market data
- `schools <id>` - School district information
- `sync` - Sync API data to local SQLite

### Raw API Access

- `stingray <subcommand>` - Direct access to all 28 Redfin Stingray API endpoints

## Output Formats

```bash
# Human-readable table (default)
redfin-pp-cli search "Seattle, WA"

# JSON for scripting
redfin-pp-cli search "Seattle, WA" --json

# Filter specific fields
redfin-pp-cli search "Seattle, WA" --json --select address,price,beds

# Dry run (show request without sending)
redfin-pp-cli search "Seattle, WA" --dry-run

# Agent mode (JSON + compact + no prompts)
redfin-pp-cli search "Seattle, WA" --agent
```

## Offline Mode

Save search results to local SQLite, then search offline:

```bash
# Save results locally
redfin-pp-cli search "Portland, OR" --save

# Search offline
redfin-pp-cli search "Portland" --offline
```

## Configuration

Config file: `~/.config/redfin-pp-cli/config.toml`

Database: `~/.local/share/redfin-pp-cli/redfin.db`

Exit codes: `0` success, `2` usage error, `3` not found, `4` auth error, `5` API error, `7` rate limited.
