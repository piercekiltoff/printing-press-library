# OpenAlex CLI research brief

OpenAlex is a free scholarly graph API covering works, authors, institutions, sources, topics, funders, and related research metadata. The official OpenAPI document is available at https://developers.openalex.org/api-reference/openapi.json and no key is required for basic live queries; an API key improves production usage/rate handling.

Useful CLI workflows center on repeatable research retrieval: search works, identify authors/institutions/sources, inspect topics, and narrow large OpenAlex responses with JSON plus field selection.
