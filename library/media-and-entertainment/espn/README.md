# espn-pp-cli

ESPN sports CLI with live scores, standings, stats, and offline search across 10 leagues.

No API key required. All endpoints are unauthenticated.

Generated by [CLI Printing Press](https://github.com/mvanhorn/cli-printing-press).

## Quick Start

```bash
# Live scores
espn-pp-cli scores nba
espn-pp-cli scores nfl --date 2026-01-12

# Standings
espn-pp-cli standings nba
espn-pp-cli standings nfl --json

# Team and player info
espn-pp-cli team Cowboys --league nfl --roster
espn-pp-cli athlete "LeBron James" --league nba

# Game details
espn-pp-cli boxscore 401671793 --league nfl
espn-pp-cli plays 401671793 --league nfl

# News and search
espn-pp-cli news nfl
espn-pp-cli search "Patrick Mahomes"
```

## Commands

### Scores and Games

| Command | Description |
|---------|-------------|
| `scores <league>` | Live and recent scores |
| `boxscore <game-id>` | Full box score for a game |
| `plays <game-id>` | Play-by-play for a game |
| `preview <game-id>` | Pre-game preview with odds and records |
| `schedule <league>` | Upcoming schedule |
| `calendar <league>` | Season calendar and week boundaries |
| `odds <league>` | Current betting lines |

### Teams and Athletes

| Command | Description |
|---------|-------------|
| `team <name>` | Team info, record, roster, schedule |
| `athlete <name>` | Player stats, game log, news |
| `injuries <league>` | Injury report across the league |
| `transactions <league>` | Recent trades, signings, releases |
| `leaders <league>` | Statistical leaders |
| `rankings <league>` | AP/coaches poll rankings (college) |

### Analysis

| Command | Description |
|---------|-------------|
| `compare <player1> <player2>` | Side-by-side stat comparison |
| `h2h <team1> <team2>` | Head-to-head history between teams |
| `sos <team>` | Strength of schedule analysis |
| `trending <league>` | Teams on hot/cold streaks |
| `dashboard` | Cross-sport scores and watchlist |

### Data Management

| Command | Description |
|---------|-------------|
| `sync` | Cache teams, athletes, standings to local SQLite |
| `search <query>` | Search ESPN and local data |
| `watch add/remove/scores` | Manage team watchlist |
| `export` | Export local data as JSON |
| `import` | Import data from JSON |
| `workflow` | Run multi-step scoring and notification workflows |

### Utilities

| Command | Description |
|---------|-------------|
| `doctor` | Verify config and API connectivity |
| `auth` | Manage authentication (optional) |

## Agent Usage

Use `--agent` for machine-friendly defaults (enables `--json --compact --no-input --no-color --yes`):

```bash
espn-pp-cli scores nba --agent
espn-pp-cli athlete "LeBron James" --league nba --agent --select name,stats
espn-pp-cli dashboard --agent
```

All commands support:
- `--json` for structured JSON output
- `--compact` for minimal fields (id, name, status)
- `--select field1,field2` to pick specific fields
- `--csv` for CSV output
- `--dry-run` to show the request without sending
- `--quiet` for bare output, one value per line

## Health Check

```bash
espn-pp-cli doctor
```

Verifies config file, API reachability, and local database status.

## Local Data

Sync caches teams, athletes, events, standings, and news into a local SQLite database for offline access and fast search.

```bash
espn-pp-cli sync
espn-pp-cli sync --resources nfl,nba
espn-pp-cli search "Lakers"
```

Default database path: `~/.local/share/espn-pp-cli/data.db`

## Troubleshooting

**API errors:** ESPN APIs are undocumented and may change without notice. Run `espn-pp-cli doctor` to check connectivity. If a specific endpoint returns errors, it may be temporarily unavailable.

**Slow responses:** Use `espn-pp-cli sync` to cache data locally, then queries hit the local database first.

**Wrong league:** Most commands require a league argument (nba, nfl, mlb, etc.). If results look wrong, check that you specified the correct league with `--league`.


```bash

```

```bash

espn-pp-cli watch add Chiefs --league nfl
espn-pp-cli watch scores
```

### Sync and search offline

```bash
espn-pp-cli sync --resources nba,nfl

```
