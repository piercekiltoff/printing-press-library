# X CLI Publish Brief

This package publishes `x-twitter-pp-cli`, a combined X API CLI that mirrors the official X v2 OpenAPI surface and adds curated browser-backed Articles commands.

Novel command coverage:

- `thread compose` splits markdown into a numbered tweet thread with atom-aware paragraph, list, and code fence handling.
- `articles-publish-md` parses markdown with YAML frontmatter and builds Draft.js `content_state` JSON for X Articles authoring.
