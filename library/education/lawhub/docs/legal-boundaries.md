# LawHub Data Boundaries

Store:
- test/attempt IDs and names
- dates, modes, scores
- section types and accuracy
- chosen/correct answer letters if exposed
- time spent and flags if exposed
- Nolan-authored review notes
- links back to LawHub

Avoid unless LSAC explicitly permits:
- full question text
- passages
- official answer explanations
- bulk copyrighted PrepTest content
