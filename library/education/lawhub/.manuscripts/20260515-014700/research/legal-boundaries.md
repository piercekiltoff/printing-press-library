# Legal / Content Boundary

This CLI is intentionally metadata-only.

Allowed to store:

- attempt IDs, test IDs, dates, modes, scores
- section IDs/types/counts
- chosen answer letters and correctness
- question type/subtype, difficulty, timing, flag state
- links back to LawHub
- user-authored review notes

Not stored:

- LSAT question stems
- passages
- answer-choice text
- official explanations
- bulk official LSAT content

Use `lawhub-pp-cli review open ...` for official content review inside LawHub.
